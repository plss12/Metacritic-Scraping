Trabajo realizado por Pedro Luis Soto Santos, correo electrónico pepoluis712@gmail.com

Este proyecto se ha realizado para la asignatura de Complemento de Bases de Datos de la Universidad de Sevilla.

El trabajo consiste en el scrapping de la web de Metacritic, más concretamente en su sección de juegos, a través de esta sección se han recuperado varios de los juegos que se encuentran en la web (limitado por el espacio disponible en la base de datos), y se han almacenado en una base de datos Atlas MongoDB distribuida, para posteriormente capacitar esta web con una amplia variedad de filtros y búsquedas para que el usuario pueda encontrar el juego que busca de una forma sencilla y rápida. Además, se ha desplegado la web en pythonanywhere para que esté continuamente disponible para los usuarios.

Para más información revisar el documento adjunto en la entrega.
